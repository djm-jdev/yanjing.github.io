测试接口：http://114.116.100.206:5006/GraphQL

之前有个示例是用来获取 所有运动员信息

query
{
  all{
    id,
    name,
    birthPlace
  }
}

这里是一个 JavaScript 客户端的示例

<!DOCTYPE html>
<html>

<head>
   <script src="https://cdn.bootcss.com/jquery/3.4.1/jquery.min.js"></script>
   <script>
      $(document).ready(function () {

         $("#btnGreet").click(function () {

            $("#greetingDiv").html('loading....');

            $.ajax({
               url: "http://114.116.100.206:5006/GraphQL",
               contentType: "application/json",
               type: 'POST',
               data: JSON.stringify({
                  query: `{all{id,name}}`
               }),
               success: function (result) {

                  $("#greetingDiv").html("<h1>" + JSON.stringify(result.data) + "</h1>");
               }
            });
         });
      });
   </script>
</head>

<body>
   <h1>Jquery Client </h1>

   <hr />
   <section>
      <button id="btnGreet">Greet</button>
      <br /> <br />
      <div id="greetingDiv"> </div>
   </section>

   <br /> <br /> <br />
   <hr />

</body>

</html>
