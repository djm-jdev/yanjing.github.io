  • 增加一条记录

测试地址：http://114.116.100.206:5006/GraphQL/

现在需要增加一条选手记录，首先我们需要先定义下需要输入输出的格式

输入模型

type PlayerInputType
{
  name:String!
  birthPlace:String
  height:String
  weightLbs:int
  birthDate:Date
}

输出模型

type PlayerType
{
  id:ID!
  name:String
  ...
  ...
}

打开最上面的测试网址进行测试

输入

mutation{
  createPlayer(player:
    {
      name:"Jarbo",
      birthPlace:"tianjin",
      height:"8'5",
      weightLbs:200,
      birthDate:"1980-05-01"}){
    id,name
  }
}

输入的字段中只有name是必须的，其他可选

返回

{
  "data": {
    "createPlayer": {
      "id": 9,
      "name": "Jarbo"
    }
  }
}
