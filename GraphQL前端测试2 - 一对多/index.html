本小节哦们将会对 选手信息player和选手比赛记录信息palyrecord 进行一对多的关联查询。

选手表 Players

序号 名 类型 解释
1 Id int 自增ID
2 Name nvarchar(MAX) 姓名
3 BirthDate datetime2 生日
4 BirthPlace nvarchar(MAX) 出生地
5 Height nvarchar(MAX) 身高
6 WeightLbs int 体重

选手比赛记录表 PlayRecords

序号 名 类型 解释
1 Id int 自增ID
2 PlayerId int 选手ID Players的关联键
3 YearName int 年份
4 CityName nvarchar(MAX) 城市
5 Scores int 得分

【选手表 Players】 中一条记录可能会有0到多条的 【选手比赛记录表 PlayRecords】表记录， PlayRecords.PlayerId 就是两个表的关联键

打开测试工具：http://114.116.100.206:5006/GraphQL/

1、查询 id = 5 的选手信息

query{
  player(id:5){
    id,
    name,
    birthDate,
    birthPlace,
    height,
    weightLbs
  }
}

返回数据

{
  "data": {
    "player": {
      "id": 5,
      "name": "Patrik Laine",
      "birthDate": "1998/4/19",
      "birthPlace": "Tampere, Finland",
      "height": "6'04",
      "weightLbs": 206
    }
  }
}

2、查询 id = 5 的选手信息，以及选手的比赛记录
